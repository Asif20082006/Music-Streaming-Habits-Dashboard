# 🎵 Music Streaming Habits Dashboard

## 📌 Overview
This project is an interactive Power BI dashboard developed to analyze music streaming habits and visualize user behavior.
## 📸 Dashboard Preview

![Music Streaming Dashboard](Screenshot%202026-07-20%20222530.png)


## 🛠️ Tools Used
- Power BI
- Microsoft Excel
- CSV Dataset

## 📊 Dashboard Features
- KPI Cards
- Bar Chart
- Pie Chart
- Line Chart
- Gauge Chart
- Slicers

## 📈 Key Insights
- Analyzed user listening behavior.
- Compared streaming patterns.
- Identified trends using interactive visualizations.
- Supported data-driven decision-making.
